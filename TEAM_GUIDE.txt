# CET4/6 智能英语教学系统 - 团队开发指南

## 环境配置

### 基本要求
- Python 3.8+
- 网络连接（用于安装依赖和调用AI API）

### 安装步骤
1. **进入项目目录**：`cd app`
2. **安装依赖**（使用国内镜像源）：
   ```bash
   pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
   ```
3. **验证依赖**：`python check_dependencies.py`
4. **运行应用**：`streamlit run app.py`
5. **浏览器访问**：http://localhost:8501

### 依赖说明
- **核心依赖**：Streamlit（Web框架）
- **AI功能**：硅基流动 API + requests
- **数据处理**：pandas
- **数据可视化**：plotly
- **API调用**：requests
- **环境管理**：python-dotenv

### 验证步骤
运行 `python check_dependencies.py` 查看所有依赖是否安装成功。如果显示所有版本信息，则依赖安装完成。

## 功能模块

| 模块 | 功能 | 说明 |
|------|------|------|
| 单词考察 | 高频小测 | 英译中选择题，优先考察错词，结合遗忘曲线单词复习 |
| 出卷 | 正常出卷 | 用户选择难度和试卷类型，系统自动控制高频词配比、语法复杂度及文本题材变量；AI生成试卷时同步产出每道题的深度解析 |
| | 个性化出卷 | 结合近期3次历史试卷（难度自适应）和错词生成个性化试卷 |
| 历史回顾 | 试卷回顾 | 查看之前做过的试卷及答案解析，主观题AI解析 |
| 学情报告 | 成绩趋势、薄弱点分析 | 可视化展示学习状况 |


## API接口说明

### 词库使用（所有人）
```python
from utils.database import db

# 获取单词（双词库配比，根据难度自动分配高频词和完整词库比例）
words = db.get_words_by_difficulty("CET4", 5, 20)  # 四级，难度5，20个单词

# 获取所有单词（支持variant参数："complete"完整词库 或 "highquality"高频词库）
all_words = db.get_all_words("CET4")  # 默认获取完整词库
all_words = db.get_all_words("CET4", "highquality")  # 获取高频词库

# 获取错词列表（优先考察这些单词）
wrong_words = db.get_wrong_words(user_id, level)
# 返回: [{"word": "abandon", "error_count": 3, "last_error_time": "..."}, ...]

# 获取已考察过的单词
tested_words = db.get_tested_words(user_id, level)
# 返回: [{"word": "steady", "is_correct": 0, "quiz_date": "2026-04-25 15:40:21"}, ...]
tested_words = db.get_tested_words(user_id, "CET4", limit=50)  # 支持limit参数
```

### B 用（正常出卷）
```python
from utils.ai_exam_pipeline import generate_exam, generate_answers_sync
from utils.call_ai import check_ai
from utils.database import db
from utils.exam_schema import (
    build_exam_paper,           # 构造完整试卷JSON结构
    build_submission,           # 构造提交/草稿状态
    build_answers_payload,      # 构造答案保存结构
    count_questions,            # 统计试卷总题数
    extract_question_bank,      # 将试卷sections拉平为题目列表
    normalize_level,            # 统一级别命名（CET4/CET6）
)

# AI调用
check_ai()                     # 检查AI接口是否可用，返回True/False

# 生成试卷（一次性生成全部6个section：写作、翻译、选词、仔细阅读x2、长篇阅读）
paper = generate_exam(level, difficulty, words, exam_type="normal_exam", generation_mode="ai")

# 生成答案与解析（同步调用，返回解析后的答案payload和带答案的试卷）
answers_payload, paper_copy = generate_answers_sync(paper)

# 数据库操作
words = db.get_words_by_difficulty(level, difficulty, 50)  # 获取50个单词作为出题参考
exam_id = db.save_exam_paper(user_id, level, paper_json)  # 保存试卷到数据库，返回exam_id
db.update_exam_paper(exam_id, paper_json=paper_json, answers_json=answers_json, status="ready")  # 更新试卷状态
```

### C 用（单词考察）
```python
from modules.word_quiz_ui import render_word_quiz_setup, render_word_quiz_interface, handle_word_quiz
from utils.call_ai import call_ai_full
from utils.database import db

# 获取单词（根据难度和数量从词库抽取）
words = db.get_words_by_difficulty(level, difficulty, word_count)

# 获取错词列表（优先考察这些单词）
wrong_words = db.get_wrong_words(user_id, level)
# 返回: [{"word": "abandon", "error_count": 3, "last_error_time": "..."}, ...]

# 获取已考察过的单词
tested_words = db.get_tested_words(user_id, level)
# 返回: [{"word": "steady", "is_correct": 0, "quiz_date": "2026-04-25 15:40:21"}, ...]
tested_words = db.get_tested_words(user_id, "CET4", limit=50)  # 支持limit参数

# 保存单词考察结果（记录用户答题情况）
db.save_word_result(user_id, level, word, is_correct, source='quiz')
# 参数：user_id用户ID, level级别, word单词, is_correct是否正确, source来源(固定'quiz')

# AI调用（生成易混淆选项）
result = call_ai_full(prompt, max_retries=2, max_tokens=2048, timeout=180)
# 参数：prompt提示词, max_retries重试次数, max_tokens最大token数, timeout超时秒数
# 返回：{"status": "success", "content": "..."} 或 {"status": "failed", "error": "..."}

# UI组件调用
level, difficulty, word_count = render_word_quiz_setup(prefix)           # 渲染难度/数量选择控件
render_word_quiz_interface(quiz_data, answers, prefix)                  # 渲染答题界面
```
